- 01/22/2021 -

Version 1 Update

Welcome to my site! This was created as a project for my class in Web Development. We are challenged to create our own web page of any content of our choosing! I chose to create a site about my French Bulldog Jax and provide any useful information needed about French Bulldogs. For example how to nurture, health problems that may occur, even help you find potential adoption and breeder pages. Now about the class, in our first week of instruction we learned Hyper Text Markup Languag (html) and how html is used to set up web pages. Then in the next week we learned abot Cascading Style Sheets also known as CSS. CSS is all about the design and style of the webpage, everything you need to make your website look good! More updates to come every 2 weeks!

- 02/05/2021 -

Version 2 Update

The past few weeks we have been learning more advanced ways to syle your webpage, for example box shadows, making boxes curved, image effects and background images. These updates were crucial to the design of the webpage itself and to make things look a lot cleaner and smoother. Also it helped to add effects pleasing to the users eyes.

- 02/20/2021 -

Version 3 Update

The last 2 weeks we have been going over and discussing Flexboxes and Tables. I personally wasn't a fan of the flexbox ideas for my website so I mainly stuck with a tables design. I found it easier to control where I wanted my information on the screen by using the table set up. The table give you access to build the amount of colums and rows you need on your page.

- 03/12/2021 -

Version 4 Update

The final 2 weeks of the Web Developement class we went over Web Forms and how to add Media to websites. I added a Question/Comment Web Form that will send me an email if anybody who visits my site has anything they would like to share with me, whether it be ways to improve my site or things they liked about it. I'm hoping having this option available to people can help me create a great website that has all the information someone would need about French Bulldogs.
